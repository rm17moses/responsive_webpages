Bootstrap

What is bootstrap?

It is a collection of pre-written code chunks in CSS, HTML, and JavaScript that allows developers to create websites more quickly than if they had to create every bit of code from scratch.

How does Bootstrap work?

Bootstrap is built to work best in the latest desktop and mobile browsers. It is designed to be responsive to mobile devices

Benefits of using Bootstrap
  • Speed of development
  • Responsiveness
  • Consistency
  • Customizable
  • Support
Examples of websites with Bootstraps
  • Spotify
  • Airbnb
  • Netflix
  • Udemy
  • Lyft

Materialize CSS

What is Materialize CSS?

It is a responsive CSS framework that includes pre-built CSS classes and JavaScript components for creating modern web pages.

How does Materialize CSS work?

Uses modal component for creating dialog boxes, confirmation messages or presenting important content to the viewers where underlying content becomes inactive until the modal window is closed.

Benefits of using Materialize CSS
  • Out of the box slick material design
  • It is responsive and easy to use
Examples of websites with Materialize CSS
  • The University of Carlifornia San Diego
  • The University of Texas at Austin
